AD-Data Analysis for FWM (Four Wave Mixing).

This Application helps you analyze your four wave mixing data taken using AD-Spectroscopy. It can export your data at particular spectrum position with respect to delay position and also can export energy spectrum and wavelength spectrum at particular delay position.
